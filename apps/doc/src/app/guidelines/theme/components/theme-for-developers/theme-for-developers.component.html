Темы в Prizm

Что такое темы и как они работают

Тема — это основа, набор правил в CSS, который определяет, как выглядит проект, построенный с помощью дизайн-системы. Она определяет не только цвета, но ещё шрифты, размер текста, отступы, скругления и тени. Можно использовать несколько тем в одном проекте: «дневную» и «ночную» для всего интерфейса или разные темы для отдельных блоков. Подробнее об этом читайте в этом гайде.

В Prizm доступно две базовые темы: light по умолчанию и dark. Для кастомизации отдельных стилевых значений, там где это допускается дизайн-системой созданы специальные переменные. Переопределять токены из палитры не рекомендуется.



Как подключить тему

Для того, чтобы подключить тему к проекту и управлять ей, используется атрибут data-prizm-theme. Его необходимо применить на контейнер, тему которому мы устанавливаем, для всего приложения в качестве контейнера рекомендуется использовать body


В проект темы Prizm подключаются с помощью PrizmThemeModule


Так же для использования тем необходимо подключить стили к вашему проекту: либо через angular.json, либо через основной стилевой файл проекта.



Переключение тем

Для переключения тем используется PrizmThemeService. Доступно переключение тем как глобально, так и на отдельный блок приложения.


Так же доступна возможность подключать реверсивные темы с использованием themeInvertedDirective. Эта директива позволяет установить на выбранный контейнер тему, обратную основной теме приложения. В дизайн-системе такой подход используется в PrizmHint, PrizmTooltip и PrizmConfirmPopup компонентах. Таким образом, весь контент этих элементов по умолчанию отображается в теме, противоположной основной. Получить значения для реверсивных тем при необходимости можно через PrizmThemeInvertedValuesService.



Как создать свою тему

*Раздел дополняется

У пользователей дизайн-системы есть несколько возможностей для расширения тем:

  1. Переопределение значений для светлой и темной темы
  2. Создание полностью кастомной темы
  3. Переопределение текущих палитр
  4. Создание новой палитры

Для того, чтобы переопределить отдельные значения для токенов, палитры или отдельной темы, файл с переопределнными значениями необходимо добававить в проект, а в основном стилевом файле добавить вызов css-функции для новых значений. Обратите внимание, что проектный файл со стилями должен быть подключен после файлов библиотеки.


Вы можете подключить свою персональную тему в Prizm. Для этого необходимо добавить в проект файлы с переопределенными токенами и палитрами. Подробнее о создании палитр и токенов можно узнать в разделе Дизайнерам.
