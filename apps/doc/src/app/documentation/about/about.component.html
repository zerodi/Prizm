Prizm

Prizm is a part of the SDK that allows you to build an application, mostly in Angular, from a set of elements and components.

Prizm includes the following `npm' packages:

@prizm-ui/core

The main library for creating Angular components and elements using Prizm.

@prizm-ui/helpers

Library to make it easy to create Angular applications. Contains Injectables Service, RxJS utilities, directives, and standard pipes.

@prizm-ui/components

Prizm UI system design components library http://prizm.zyfra.com

@prizm-ui/charts

Library of components for work with charts

@prizm-ui/install

Package installer @prizm-ui/* , Angular schematic ng-add
